Supermarket Compare (Israel)

Overview
This is a static website for comparing product prices across supermarkets in Israel. It uses sample data and can be hosted with Firebase Hosting.

Features
- Search and filter by category and max price
- Toggle supermarkets to compare
- Best price highlights and comparison cards

How to add your data
Update the product list in public/app.js with real products and prices.

Firebase Hosting
This project includes firebase.json for Firebase Hosting. Create a Firebase project and add a .firebaserc file with your project id when you are ready to deploy.
